# Sitio Web Fundación Cultura Local

Proyecto desarrollado como parte de la **Práctica Profesional I** de Ingeniería Civil Informática en la **Universidad del Bío-Bío**.  
El objetivo es implementar un sitio web funcional y sostenible para la Fundación Cultura Local, fortaleciendo sus canales de comunicación digital y apoyando la difusión de actividades culturales y comunitarias.

---

## Objetivos del Proyecto
- Desarrollar e implementar un sitio web desde cero.
- Entregar una herramienta práctica para la difusión y visibilización de actividades culturales.
- Incluir funcionalidades esenciales como formularios de inscripción, calendario de eventos y secciones informativas.
- Asegurar accesibilidad, usabilidad y sostenibilidad del sitio.

---

## Actividades Principales
1. Levantamiento de requerimientos junto al equipo de la Fundación.
2. Diseño de la estructura frontend considerando usabilidad y estética.
3. Diseño de la base de datos y definición de funcionalidades clave.
4. Desarrollo backend con conexión a base de datos y APIs.
5. Desarrollo frontend con diseño responsivo e interacción moderna.
6. Pruebas funcionales y de compatibilidad en distintos dispositivos.
7. Capacitación básica al equipo de la Fundación para administrar contenidos.

---

## Tecnologías Utilizadas
- Frontend: HTML5, CSS3, JavaScript (framework/librería a definir según requerimientos).
- Backend: Node.js / Express (o equivalente).
- Base de Datos: MySQL / PostgreSQL.
- Control de versiones: Git + GitHub.
- Herramientas adicionales: pruebas en navegadores, diseño responsivo, APIs REST.
